What to do?

Place the nadeshiko folder in your desktop, then open it

Make a shortcut of Yuru Camp Launcher.bat by right click, go to properties then change the icon to shiko.ico.

Configure the sources.bat file with path to the episodes

Place it to desktop then you're done!

for best window size and experience, go to "Font" tab in properties, then select
Consolas with size 16, click apply. :D

Enjoy!
